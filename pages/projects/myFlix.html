Objective:

To build the server­side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies. Then using React, build the client-side for the application based on its existing server-side code.

Key Features:

  • Returns a list of ALL movies to the user.
  • Returns data about a single movie by title to the user.
  • Returns data about a genre by name/title.
  • Returns data about a director by name.
  • Allows new users to register.
  • Allows users to update their user info.
  • Allows users to add a movie to their list of favorites.
  • Allows users to remove a movie from their list of favorites.
  • Allows existing users to deregister.
